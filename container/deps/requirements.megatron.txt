# Third-party Python dependencies for the megatron runtime image. Installed
# with --no-deps so upstream nvcr.io/nvidia/pytorch's solve stays intact.
# Megatron-LM's own pyproject pulls the rest of its inference deps via
# `uv pip install -e /opt/megatron-lm` in the Dockerfile.
# Sorted alphabetically per pre-commit requirements-txt-fixer.

# Megatron's InferenceClient wire protocol (DEALER↔ROUTER + per-step partial

# them; we install them here at fixed lower bounds for the cases where the
# NGC PyTorch base ships an older copy.

# Tokenizer used by both the Dynamo frontend (when it sees the served-model-name)
# and Megatron-LM's HuggingFaceTokenizer flag.

# Required by ai_dynamo_runtime (companion to msgspec).
